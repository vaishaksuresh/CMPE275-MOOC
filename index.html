Welcome to CMPE - 275 MOOC Project's Common Repository.

Architecture Idea : 1

  1. The application has 2 different components, A Server and a Client.
  2. Each team builds their own server and client.
  3. Every server provides a base set of services.
  4. The URL for the request to these services are agreed upon by everybody.
  5. The JSON response+ to this request is also agreed upon by everybody.
  6. The servers can provide additional services and the clients are free to use/not use them.
  7. Now the clients are interchangeable, as long as the base functions are the only functions used.
